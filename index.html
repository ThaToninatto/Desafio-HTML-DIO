Tag h1 a h6

(Índice)

Utilizadas para títulos e possuem 6 tamanhos (h1 até h6), sendo h1 o maior título e h6 o menor título.

Tag p

(Índice)

Utilizada para parágrafos.

Tag mark

(Índice)

Utilizada para destacar partes do texto, como se estivessem grifadas.

Tag small

(Índice)

Utilizada para diminuir o tamanho da fonte.

Tag i

(Índice)

Utilizada para deixar uma palavra ou trecho em itálico.

Tag u

(Índice)

Utilizada para deixar uma palavra ou texto sublinhado.

Tag strong

(Índice)

Utilizada para deixar uma palavra ou texto em negrito.

Tag ol

(Índice)

Utilizada para criar listas ordenadas.

Tag ul

(Índice)

Utilizada para criar listas não ordenadas.

Tag li

(Índice)

Utilizada para representar um item de uma lista ordenada ou não ordenada.

Tag a

(Índice)

Utilizada para inserir hiperlinks em uma página. Pode ser usada para navegação entre páginas, navegação na mesma página e inserção de endereços URL.

Tag hr

(Índice)

Utilizada para fazer a quebra entre seções da mesma página.

Tag sub

(Índice)

Utilizada para escrever palavras ou textos subscritos, com letras menores e um pouco abaixo do resto das palavras.

Tag sup

(Índice)

Utilizada para escrever palavras ou textos sobrescritos, com letras menores e um pouco acima do resto das palavras.

Tag blockquote

(Índice)

Utilizada para citações

Tag font

(Índice)

Utilizada para definir o tamanho e a cor da fonte.

Tag del

(Índice)

Utilizada para representar uma parte do texto que foi excluída.

Tag abbr

(Índice)

Utilizada para abreviações.
